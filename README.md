# estatequoter
A fast, user-friendly estate liquidation &amp; planning quote platform. Built for executors, families, and advisors.

EstateQuoter.com

This is the website for EstateQuoter, a simple platform for requesting estate liquidation and estate planning quotes.
The site is a single static HTML file designed for fast deployment on Netlify.
